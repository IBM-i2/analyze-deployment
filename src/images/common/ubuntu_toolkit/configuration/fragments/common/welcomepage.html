Welcome message

This HTML file is a placeholder for content that you can display to Analyst's Notebook users when they log in to your deployment of i2 Analyze.

To display this page to users at login, set EnableWelcomePage=true in the DiscoServerSettingsCommon.properties file (or the analyze-settings.properties file, if you have one).

Typical content that you might present here includes:

  • Your corporate branding, such as a logo or a color scheme
  • A description of your i2 Analyze deployment, including the features that it provides
  • Links to sections of the i2 documentation that explain the features you deployed
  • Links to your documentation that gives detail on using i2 Analyze in your organization

Analyst's Notebook displays this content in a dialog that also contains a Close button and an option for users to hide the dialog in future.
